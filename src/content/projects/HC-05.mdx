## â“ Problem
1. Pengendalian perangkat elektronik yang masih dilakukan secara manual.
2. Tidak adanya sistem monitoring untuk mengetahui status perangkat apakah menyala atau mati.
3. Kesulitan dalam mengatur jadwal pengendalian perangkat.

---

## ğŸ¯ Goal
- Menyediakan aplikasi yang mudah digunakan untuk mengendalikan perangkat elektronik berbasis Arduino UNO.
- Memberikan fitur monitoring status perangkat secara real-time.
- Memungkinkan pengaturan jadwal otomatis untuk pengendalian perangkat.

---

## ğŸ’¡ Solution
Sistem ini terdiri dari dua bagian utama:

1. **Perangkat Keras:**
   - Mikrokontroler: Arduino UNO
   - Modul Bluetooth untuk komunikasi nirkabel
   - Relay untuk mengendalikan lampu
   - Sensor (opsional) untuk monitoring kondisi lingkungan

2. **Perangkat Lunak:**
   - Aplikasi mobile (Android) untuk antarmuka pengguna
   - Firmware Arduino untuk mengendalikan perangkat keras
   - Sistem jadwal otomatis (berbasis waktu sistem)

---

## ğŸ”„ User Flow
1. **Buka Aplikasi**
2. **Koneksi Bluetooth**
   - Pilih perangkat Arduino UNO
   - Hubungkan
3. **Lihat Status Perangkat**
   - Status perangkat: Menyala / Mati
   - Kondisi lingkungan (jika ada sensor)
4. **Atur Jadwal**
   - Waktu hidup dan mati
   - Tombol Simpan / Reset Jadwal
5. **Kendali Manual**
   - Tombol "Nyalakan Sekarang" / "Matikan Sekarang"
