**Smart Feeder Ikan** adalah proyek IoT berbasis ESP32 yang dirancang untuk memberi makan ikan secara otomatis dan terjadwal. Sistem ini dapat dipantau dan dikendalikan melalui antarmuka berbasis web yang ringan dan responsif.

## â“ Problem
1. Pemilik kolam sering lupa atau terlambat memberi makan ikan.
2. Tidak adanya sistem monitoring jarak jauh status perangkat.
3. Kesulitan mengatur jadwal otomatis dan manual pemberian makan.

---

## ğŸ¯ Goal
- Menyediakan antarmuka mudah digunakan untuk mengatur jadwal makan ikan.
- Memberikan fitur kendali manual dari jarak jauh melalui web.
- Memantau status perangkat (online/offline, stok pakan, waktu terakhir makan).

---

## ğŸ’¡ Solution
Sistem ini terdiri dari dua bagian utama:

1. **Perangkat Keras:**
   - Mikrokontroler: ESP32
   - Servo untuk membuka pakan
   - RTC (Real-Time Clock) untuk akurasi waktu
   - LCD I2C untuk tampilan lokal
   - Sensor stok pakan (opsional)

2. **Perangkat Lunak:**
   - Web dashboard (PHP, JS)
   - Komunikasi MQTT antara server dan ESP32
   - Sistem jadwal otomatis (berbasis RTC)
   - Fitur kontrol manual dari web

---
